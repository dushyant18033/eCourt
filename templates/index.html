Honour : An eCourt Platform

An eCourt Portal for managing court cases online with the power of databases.

About Honour

The objective of the project is to design and develop a system/platform to ease the overall process experienced by any user involved in any judicial case. This platform aims to simplify the process of filing a case and then track it in its various stages.

The platform would be a web based application which would provide an end to end solution for any user ranging from any client filing a case to police and other stakeholders. The application would ensure transparency amongst various users and thus regulate the entire pipeline involved in serving justice to a citizen.
